## Sendsoknad

# Aksessloggene
Aksesslogger til sendsoknad kan finnes i [Kibana](https://logs.adeo.no) ved å søke med:
```
+type:accesslog +referer: https\:\/\/tjenester.nav.no\/soknad<navn>*
```

evt med:

```
+type:accesslog +path:\/sendsoknad*
```
